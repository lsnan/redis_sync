# redis_sync

### 通过 redis monitor 命令实现 redis 到 redis 的同步迁移

## 使用方法

## Features

- 将 redis monitor 获取到的相关命令同步在目标 redis 库执行
- 将 redis monitor 获取到的相关命令输出到指定文件 

## 注意事项

- 源 redis 连接异常, 程序会直接退出
- 目的 redis 连接异常或写入异常, 不退出程序
- 目的 redis 连接异常或写入异常, 丢弃当前条目, 继续下一条记录的写入
